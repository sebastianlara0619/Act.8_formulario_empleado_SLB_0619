# act8_formulario_empleados_0619

Prompt
Plataforma antigravity
Lenguaje: Dart
Proyecto: Datos empleado
Descripción de proyecto
Lenguaje de programación: dart flutter, crear un formulario para capturar 4 datos de un empleado y un botón para guardar dichos datos, crear una clase empleado con atributos(id, nombre, puesto, salario) , inicializar un diccionario vacío llamado datosempleado. Que el id sea autonumérico, conforme se vayan registrando se guarden en el diccionario, crear archivos, main.dart, claseempleado.dart, diccionarioempleado.dart, verempleados.dart, crear un agente que se llame guardardatosdiccionario.dart para modificar el archivo diccionarioempleado.dart nivel principiante, formulario atractivo y elegante, buena distribución de los widgets, utilizar rutas nombradas para navegar entre pantallas, inicio.dart con ella con dos botones, uno para capturar empleados  capturaempleados.dart, y otro botón para ver empleados, verempleados.dart, que se totalmente funcional.
